Git Шпаргалки

Команда Що буде якщо виконати
git init

Ініціалузвати сховище для нового або існуючого проекту

git clone

Скопіювати проект із репозиторія

git status

Поточний статус гілки та комітів
git add Додає зміни в нашому робочому каталозі
git commit

Зберігаємо Ваші зміни

git commit -m "Здійснили такі от зміни"
git push

Відправити ваші зміни на репозиторій
git pull Отримати Ваші зміни із серверного репозиторію
git branch

Показати всі гілки на Вашому гіт проекті

git checkout

Змінити гілку

table-branch -> master

git checkout master

git checkout -b table-2
